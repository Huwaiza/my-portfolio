How should the theme toggle work?

Light theme support — pick how the user switches between dark and light

A

Toggle button in navbar

A small sun/moon icon sits in the top-right of the nav. One click switches theme. Remembers preference via localStorage. Most common pattern — users know exactly where to find it.

B

Follows OS preference automatically

Uses prefers-color-scheme media query. No button needed — dark mode users get dark, light mode users get light. Zero UI clutter. No user control though.

C

Both — OS default + manual override

Starts with OS preference, but user can toggle and override. Preference saved to localStorage. Best of both worlds — respects system settings but gives control.
